# Simbioz Academy Theme (Odoo 17)

Модуль бренд-стилю Simbioz Academy для Odoo 17 Website Builder: фірмові сніпети,
модель курсів із бекенд-меню та дизайн-токени за SIMBIOZ UI KIT.

## Що всередині

| Компонент | Опис |
|---|---|
| **7 сніпетів** для Website Builder | Hero, Переваги, Кроки, Рівні навчання, Курси (динамічний), Відгуки, CTA-стрічка — усі в категорії «Structure» з префіксом «Simbioz» |
| **Модель `simbioz.course`** | Курси редагуються в меню **Simbioz Academy → Курси** без правок коду; поля: назва, рівень, опис, платформа практики, тривалість, вартість, публікація |
| **6 стартових курсів** | Бізнес-аналітик, Системний аналітик, Бухгалтер, CRM-маркетолог, Project Manager, Інтегратор (`noupdate` — ваші правки не перезапишуться) |
| **Динамічний сніпет «Курси»** | JS-віджет підтягує опубліковані курси через `/simbioz_academy/courses` (JSON, public) — зміни в бекенді одразу видно на сайті |
| **Дизайн-токени** | Sansation (заголовки) + Montserrat (текст), кольори UI-кіту: navy `#333552`, yellow `#FAB62C` (тільки прогрес/акценти), бейджі рівнів за таблицею кіту |

## Встановлення

1. Скопіюйте папку `simbioz_academy_theme` в один із каталогів `addons_path`
   вашого Odoo 17 (наприклад, `custom_addons/`).
2. Перезапустіть Odoo та оновіть список застосунків:
   **Apps → Update Apps List** (потрібен режим розробника).
3. Знайдіть **Simbioz Academy Theme** і натисніть **Activate**.

## Використання

### Сніпети
Відкрийте будь-яку сторінку сайту → **Edit** → у панелі блоків (категорія
**Structure**) знайдіть блоки з префіксом **Simbioz** і перетягніть на сторінку.
Тексти й кнопки редагуються прямо в білдері.

### Курси
- Редагування: меню **Simbioz Academy → Курси** (потрібна група Website Designer).
- Сніпет «Simbioz — Курси» показує лише записи з увімкненим «Опубліковано на сайті»,
  відсортовані за полем «Послідовність» (drag-and-drop у списку).
- Блок карток у сніпеті позначений `o_not_editable` — його не можна редагувати
  в білдері, бо вміст рендериться з бекенду.

### Форма заявки
Кнопки CTA ведуть на стандартну сторінку **/contactus** (вбудований механізм
website form → повідомлення на email компанії). За потреби напрям заявок можна
змінити в налаштуваннях сайту або замінивши `href` кнопок у білдері.

### Логотипи
У `static/src/img/` лежать готові файли для налаштувань сайту
(**Website → Configuration → Settings**):
- `logo_S_Academy.png` — логотип для світлого хедера;
- `Simbioz_logo-on-dark_UA.png` — версія для темного футера.

## Примітки

- Модуль **не** є темою в сенсі перемикача тем Odoo (для цього ім'я мало б
  починатися з `theme_`); він встановлюється як звичайний website-застосунок
  і працює поверх будь-якої активної теми.
- Кольори застосовуються лише до сніпетів модуля (префікс класів `simbioz-`),
  тож конфліктів зі стилями активної теми немає.
